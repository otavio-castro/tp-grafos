## ?? Exemplos de Grafos

### Exemplo 1: Grafo Padrão (Árvore)
```
        A
      / | \
     B  C  D
    / \  |  |
   E  F  G  H
   |     |
   I     J
```

**BFS partindo de A:**
- Nível 0: A
- Nível 1: B, C, D
- Nível 2: E, F, G, H
- Nível 3: I, J

**Ordem de visitação:** A ? B ? C ? D ? E ? F ? G ? H ? I ? J

### Exemplo 2: Rede Social
```
Alice -- Bob -- Eve -- Ian
  |       |
Carol   Frank
  |
Grace
  |
Jack

Dave -- Henry
```

**BFS partindo de Alice:**
- Nível 0: Alice
- Nível 1: Bob, Carol, Dave
- Nível 2: Eve, Frank, Grace, Henry
- Nível 3: Ian, Jack

**Aplicação:** Encontrar grau de separação entre pessoas

### Exemplo 3: Mapa de Cidades
```
    SP -------- RJ
    |  \         |
    |   \        |
   BH  Campinas  Vitoria
    |      |
Brasilia Sorocaba
    |
 Goiania
```

**BFS partindo de SP:**
- Nível 0: SP
- Nível 1: BH, Campinas, RJ
- Nível 2: Brasilia, Sorocaba, Vitoria
- Nível 3: Goiania

**Aplicação:** Encontrar caminho mais curto entre cidades

## ?? Casos de Uso do BFS

### 1. Menor Caminho (Grafo não-ponderado)
BFS garante encontrar o caminho com **menor número de arestas** entre dois vértices.

**Exemplo:**
```
Encontrar caminho de SP até Goiania:
SP ? BH ? Brasilia ? Goiania (3 passos)
```

### 2. Nível de Separação
Descobrir a distância entre dois nós.

**Exemplo em Rede Social:**
```
Distância entre Alice e Ian:
Alice (nível 0) ? Bob (nível 1) ? Eve (nível 2) ? Ian (nível 3)
Resultado: 3 graus de separação
```

### 3. Verificar Conectividade
Determinar se um grafo é conectado.

**Como:** Se BFS visita todos os vértices, o grafo é conectado.

### 4. Encontrar Componentes
Identificar grupos desconectados no grafo.

**Exemplo:**
```
Grafo com 2 componentes:
Componente 1: A, B, C
Componente 2: D, E
```

## ?? Criando Seu Próprio Grafo

### Formato do Arquivo
```
# Comentários começam com #
# Formato: vertice:adjacente1,adjacente2,adjacente3

# Seus vértices
V1:V2,V3
V2:V1,V4
V3:V1,V5
V4:V2
V5:V3
```

### Exemplo Prático: Sistema de Metrô
Crie `Data/metro.txt`:
```
# Linha Azul
Se:Tiradentes,Bresser
Tiradentes:Se,ArmeniaTiradentes
Bresser:Se,Belem

# Linha Vermelha
Se:Republica,PedroII
Republica:Se,BarraFunda
PedroII:Se,Anhangabau
```

### Exemplo Prático: Estrutura de Pastas
Crie `Data/pastas.txt`:
```
# Hierarquia de diretórios
C:Program_Files,Users,Windows
Program_Files:Visual_Studio,Git
Users:Public,Usuario1
Visual_Studio:2022,Extensions
```

## ?? Personalizando o Código

### Adicionar Novo Arquivo no Menu
Edite `Program.cs`:
```csharp
static string SelecionarArquivoGrafo()
{
    var opcao = AnsiConsole.Prompt(
        new SelectionPrompt<string>()
            .Title("[yellow]Selecione o arquivo de grafo:[/]")
            .AddChoices(new[]
            {
                "Grafo Padrão",
                "Rede Social",
                "Mapa de Cidades",
                "Metrô",           // ? Adicione aqui
                "Pastas"           // ? E aqui
            }));

    return opcao switch
    {
        "Grafo Padrão" => "Data/grafo.txt",
        "Rede Social" => "Data/grafo_rede_social.txt",
        "Mapa de Cidades" => "Data/grafo_cidades.txt",
        "Metrô" => "Data/metro.txt",        // ? Adicione aqui
        "Pastas" => "Data/pastas.txt",      // ? E aqui
        _ => "Data/grafo.txt"
    };
}
```

## ?? Exercícios Propostos

### Exercício 1: Criar Grafo Direcionado
Modifique o código para suportar grafos direcionados (arestas com direção).

### Exercício 2: Implementar DFS
Adicione Busca em Profundidade (Depth-First Search) seguindo a mesma arquitetura.

### Exercício 3: Caminho Mais Curto
Modifique BFS para retornar o caminho completo entre dois vértices.

### Exercício 4: Grafo Ponderado
Adicione pesos às arestas e implemente algoritmo de Dijkstra.

### Exercício 5: Visualização Gráfica
Integre uma biblioteca de grafos para visualização gráfica (GraphViz, QuickGraph).
